# SAMD Doctor Directory

## System Architecture
This project consists of three parts:

1.  **Backend** (`backend/`): Flask API handling search, admin, and map logic.
2.  **Web** (`web/`): Public-facing doctor search and map.
3.  **Mobile App** (`mobile_app/`): Flutter app for Android/iOS.
4.  **Admin Panel** (`admin_panel/`): Local HTML tool for managing data.

## Deployment
See `DEPLOYMENT_PYTHONANYWHERE.md` for hosting instructions.

## Development
*   **Run Backend**: `python backend/app.py`
*   **Run Mobile App**: `cd mobile_app && flutter run`
